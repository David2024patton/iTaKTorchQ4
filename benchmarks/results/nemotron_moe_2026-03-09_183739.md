## Analysis

### MoE Shared Memory Hypothesis

nemotron-3-nano has 30B total parameters but only 3B active per forward pass.
This MoE architecture means most weights are 'cold' during inference.
The question: does placing cold expert weights in shared GPU memory (system RAM
accessible via PCIe) perform differently than dedicated VRAM or pure CPU?

### Key Finding

- **GPU Max (shared memory spill) vs CPU Only:** 0.76x speedup
- CPU Only: 15.8 tok/s
- GPU Max (spill): 12 tok/s
